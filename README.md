# 🔥 MeMath — เว็บฝึกโจทย์คณิตศาสตร์ (ครู/นักเรียน)

แอปฝึกคณิตแบบเกม: ครูเป็นแอดมินเพิ่มนักเรียน + สั่งภารกิจรายวัน, นักเรียนทำโจทย์เก็บแต้มและสะสม "ไฟต่อเนื่อง"

## ฟีเจอร์
- **ครู (แอดมิน)**: เพิ่ม/ลบนักเรียน (นักเรียนสมัครเองไม่ได้), รีเซ็ตรหัส, คลังโจทย์ CRUD พร้อมพรีวิวสมการสด,
  สร้างภารกิจรายวัน (เลือกวันที่ + ขอบเขตชั้น/ห้อง/ปี/เทอม, เลือกโจทย์เองหรือสุ่มจากคลังตามดาว/หัวข้อ, ตั้งเวลาต่อข้อได้),
  ดูคะแนนนักเรียนละเอียด + กราฟไฟต่อเนื่อง
- **นักเรียน**: แดชบอร์ดภารกิจวันนี้, เล่นทีละข้อแบบเกม (ช้อย/คีย์แพดตัวเลข), จับเวลา/ไม่จับเวลา,
  เฉลยทันทีพร้อมคำอธิบาย, แต้ม + โบนัสไฟต่อเนื่อง, หน้าสถิติ
- **สมการ**: เก็บเป็น LaTeX ในฐานข้อมูล (คั่นด้วย `$...$`) แสดงผลด้วย KaTeX
- **ไฟต่อเนื่อง (streak)**: ทำทุกวันไฟยิ่งร้อน (แดงเข้ม→ส้ม→เหลือง), เว้นวันไฟดับแล้วเริ่มก่อใหม่

## รัน
```bash
npm install
npm run dev        # http://localhost:3000 (ใช้ SQLite ./data/dev.db)
```
บัญชีครูเริ่มต้น: `admin` / `admin1234` (เปลี่ยนใน production)

## DB
เลือก backend อัตโนมัติ — มี `POSTGRES_URL` (Supabase) ใช้ Postgres, ไม่มีใช้ SQLite (ดู `lib/db.ts`)
คลังโจทย์ตั้งต้น seed จาก `lib/seed-problems.json` (เพิ่ม `SEED_VERSION` เมื่อแก้)

## Deploy
Vercel + Supabase (โปรเจกต์แยกจาก PyLearn) — ตั้ง `AUTH_SECRET`, `POSTGRES_URL`
